# Contributing

## Setup

```bash
pnpm install
```

## Development

```bash
pnpm dev
```

## Verification

```bash
pnpm verify   # lint + typecheck + translation lint + tests
```

## Commit format

Conventional commits are enforced:

```
feat: add months view
fix: correct leap-year week calculation
chore: update dependencies
```

## Branch

Work on feature branches off `main`. Direct commits to `main` or `master` are discouraged.
